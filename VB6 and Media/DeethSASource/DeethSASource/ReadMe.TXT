Deeth Spectrum Analyzer v1.0
----------------------------

This is the Visual Basic source for a simple audio
spectrum analyzer... reads data from the soundcard's
input, FFTs it, and displays it.

A compiled version of this application is available
on the Deeth website in the DeethWare section.

Once again, thanks go to Don Cross for providing the
original FFT code.  His website is:
http://www.intersrv.com/~dcross/fft.html
